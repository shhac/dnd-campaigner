# Save Point Checklist

Detailed checklist for each save point type.

## End of Combat Save

```markdown
### story-state.md
- [ ] Update party HP for each character
- [ ] Note spell slots/abilities expended
- [ ] Record combat outcome
- [ ] Update any NPC status (fled, captured, killed)
- [ ] Note loot acquired (if any)
- [ ] Update quest progress if combat was objective-related

### party-knowledge.md
- [ ] Add combat to "Recent Events"
- [ ] Update any NPC relationships changed by combat
- [ ] Note any information learned during fight
```

## End of Scene Save

```markdown
### story-state.md
- [ ] Update current location
- [ ] Note time passage
- [ ] Record any changes to NPC dispositions
- [ ] Update active quest status
- [ ] Note any secrets revealed or discovered

### party-knowledge.md
- [ ] Update "Current Situation" section
- [ ] Add new location to "Locations Visited" if first visit
- [ ] Update any NPC relationship changes
- [ ] Add discovered information to appropriate section
```

## Major Discovery Save

```markdown
### story-state.md
- [ ] Record what was discovered
- [ ] Note how it affects active plots
- [ ] Update any NPC knowledge/awareness
- [ ] Adjust upcoming events if discovery changes things

### party-knowledge.md
- [ ] Add discovery to "Facts Learned" or relevant section
- [ ] Update quest information if relevant
- [ ] Note who knows what (if different characters learned different things)
```

## After NPC Conversation Save

```markdown
### story-state.md
- [ ] Update NPC disposition toward party
- [ ] Record any deals, promises, or agreements
- [ ] Note information NPC now knows about party
- [ ] Update NPC's hidden agenda if affected

### party-knowledge.md
- [ ] Add NPC to "NPCs Met" if first meeting
- [ ] Update relationship status
- [ ] Record key information exchanged
- [ ] Note any quests given or updated
```

## End of Session Save

```markdown
### story-state.md
- [ ] Full current situation summary
- [ ] All active quests with progress
- [ ] Party resources (HP, gold, consumables, spell slots)
- [ ] All NPC dispositions and locations
- [ ] Upcoming planned events
- [ ] Current scene: {NNN} - {scene-slug}
- [ ] Date: {date}
- [ ] Cliffhanger/stopping point note

### party-knowledge.md
- [ ] Current situation from party's view
- [ ] Active quests and objectives
- [ ] Recent session summary (2-3 paragraphs)
- [ ] Key NPCs and relationships
- [ ] Important locations
```
